# raw_ippi_1956_2024



Este dataset contiene datos históricos del Industrial Product Price Index (IPPI) de Canadá desde 1956 hasta 2024., DAX , POWER BI, EXCEL, Limpieza de Datos, variacion 

El IPPI mide los cambios en los precios que los productores canadienses reciben por bienes industriales manufacturados. Los datos permiten analizar tendencias inflacionarias, evolución de precios industriales y comportamiento económico a lo largo de más de seis décadas.

Se desarrolló un dashboard analítico utilizando datos históricos del Industrial Product Price Index (IPPI) de Canadá entre 1956 y 2024, con el objetivo de analizar tendencias inflacionarias, evolución de precios industriales y variaciones económicas a lo largo del tiempo.

En el presente análisis se trabajó con datos históricos del Industrial Product Price Index (IPPI) de Canadá correspondientes al período 1956–2024.

Durante el desarrollo del proyecto se analizaron los siguientes KPIs principales:

Variación anual del índice industrial
Variación mensual
Productos con mayor inflación
Productos con mayor caída de precios
Promedio histórico del IPPI
Máximo histórico alcanzado
Crecimiento acumulado desde 1956

Además, se desarrollaron visualizaciones interactivas para identificar tendencias históricas y comportamientos inflacionarios relevantes dentro del sector industrial.

Uno de los hallazgos más importantes observados en el dashboard corresponde al incremento significativo del indicador “Heavy fuel oils” durante el año 2008, comportamiento que coincide con la crisis financiera global y la fuerte volatilidad de los precios energéticos registrada en ese período.

El proyecto permitió aplicar técnicas de análisis temporal, visualización de datos y construcción de KPIs mediante Power BI, con el objetivo de transformar datos históricos en información útil para la interpretación económica y el análisis de tendencias industriales.
